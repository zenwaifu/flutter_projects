# mydiary
My Diary is a personal journaling app designed to help you capture and organize your daily thoughts, memories, and experiences in a secure and visually appealing way. With a soft pink pastel-themed interface, My Diary combines simplicity with functionality, making it perfect for anyone who wants to maintain a consistent journaling habit.

# functions
create, view, edit and delete entries, image support, search functionality, pagination
